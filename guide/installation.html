Установка

Примечание: Vue.js не поддерживает IE8 и ниже.

Самостоятельная установка

Просто загрузите и добавьте на станицу с помощью тега script Vue будет зарегистрирована как глобальная переменная.

Версия для разработки234.51kb, много комментариев и сообщений отладки/предупреждений.

Production версия70.95kb минифицированная / 23.10kb сжатая

CDN

Также доступна на jsdelivr и cdnjs (требуется некоторое время для обновления, последняя версия может быть недоступна).

CSP-совместимый билд

Некоторые окружения, такие как Google Chrome Apps, навязывают Content Security Policy (CSP) и не позволяют использовать new Function() для обработки выражений. В этом случае вы можете использовать CSP-совместимый билд.

NPM

Bower

Загрузчики модулей AMD

Версия standalone или установленная с помощью Bower обёрнуты с помощью UMD, поэтому могут быть использованы в качестве AMD модуля.
